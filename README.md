# YAP
Yet Another Player
Yap Ain't a Player

## 主要功能
- 音乐播放和暂停
- 乐曲间的切换
- 显示通知(通知中可停止、开始、下一首、上一首切换)
- 调用系统均衡器
- 查找本地＞=30s的音频文件以播放
- 使用拖动条切换进度
- 电话接入时，中止音乐播放

## 具体使用方法

- 打开YAP
- 赋予其读写权限
- 点击列表音频,enjoy it
- 停止、开始、下一首、上一首切换
- 点击右上角按钮，设置系统均衡器
- 在通知栏，停止、开始、下一首、上一首切换
- 在通知栏，滑动关闭当前通知
